Experimental VVVV Plugin to access touch data on a Microsoft Surface without
the use of any services in between - like a TUIO proxy or alike.

Use at own risk and make sure you have the SurfaceSDK 2.0 installed and run at least Windows Vista!

For Downloads and details, go to:

http://msdn.microsoft.com/en-us/library/ff727815.aspx

https://www.microsoft.com/en-us/download/details.aspx?id=26716
